═══════════════════════════════════════════════════════════════════════════════
📋 O QUE FAZ O INSTALADOR
═══════════════════════════════════════════════════════════════════════════════

1. Detecta distribuição Linux (Ubuntu/Debian/CentOS/Arch)
2. Instala Python 3.8+, pip, curl, jq, wget
3. Baixa TODOS os arquivos do repositório Git
4. Cria ambiente virtual Python em /opt/openconnect-gateway/venv/
5. Instala dependências (requests, PyYAML, urllib3)
6. Cria usuário dedicado "openconnect"
7. Configura serviços systemd (gateway + supervisor)
8. Pergunta configurações básicas (URL, store_id, threads)
9. Inicia o gateway automaticamente

═══════════════════════════════════════════════════════════════════════════════
🤖 AGENTE SUPERVISOR (AUTO-UPDATE)
═══════════════════════════════════════════════════════════════════════════════

• Monitora o gateway a cada 60s (health check)
• Reinicia automaticamente se falhar
• Verifica atualizações no Git a cada 5 minutos
• Cria backup ANTES de qualquer update
• Valida sintaxe Python antes de aplicar
• Restaura backup automaticamente se update falhar
• Rastreia: gateway.py, supervisor.py, requirements.txt, scripts/*.sh

═══════════════════════════════════════════════════════════════════════════════
🏗️ ARQUITETURA PARA LARGA ESCALA
═══════════════════════════════════════════════════════════════════════════════

• Multi-threading com ThreadPoolExecutor
• Retry automático com backoff exponencial
• Pool de conexões HTTP persistente
• Graceful shutdown (SIGTERM/SIGINT)
• Métricas em tempo real (JSON)
• Configuração hot-reload (sem reiniciar)
• Rate limiting e HMAC signature
• Sandbox systemd (ProtectSystem, NoNewPrivileges)
